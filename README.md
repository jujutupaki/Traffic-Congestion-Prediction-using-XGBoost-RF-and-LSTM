# Traffic Congestion Prediction using XGBoost RF and LSTM
This is the proof-of-concept code for our thesis "Predicting Traffic Congestion Levels under Different Weather Conditions using Machine Learning Approaches"
by: Baleña, Ruma, Ruma

## 🔗 UI Dashboard for the ML project:

https://brr-traffic.streamlit.app/

## 🎯 Purpose

The goal of this repo is to:
- Showcase the source code
- Give an overview of the dataset and the dataset cleaning conducted
- Present the feature importance and selection
- Evaluate and compare the final, hypertuned ML models

## 📁 File Details

This repo contains the following:
- `Data_Cleaning.ipnyb`: Data preparation and EDA
- `GPU_Data_Modelling_Final.ipynb.`: Cross-validation Split, Feature Selection, Baseline Models and the Final Hypertuned Models
- `README.md`: Project-specific summary
- `Resources/`: The raw, cleaned, and merged datasets
